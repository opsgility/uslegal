# Microsoft 365 Copilot for Legal Services Sales

Welcome to the US Legal Copilot Workshop. This hands-on session teaches you how to use Microsoft 365 Copilot across Outlook, Excel, Teams, and Word to accelerate your sales workflows — from prospecting into new law firms to expanding existing accounts.

## Getting Started

Complete these three steps before the first exercise. For the full lab setup including the Copilot prompting framework and workshop overview, see the **[Lab Setup](USLegal-Course/00-lab-setup.md)**.

### Step 1: Download the resource files

1. Select the link below to download the resource files:

    **[Download ResourceFiles.zip](https://github.com/opsgility/uslegal/raw/refs/heads/main/USLegal-ResourceFiles/ResourceFiles.zip)**

2. Once the download completes, open **File Explorer** and navigate to your **Downloads** folder.
3. Right-click **ResourceFiles.zip** and select **Extract All...**.
4. Choose a location (your **Downloads** folder or **Desktop** is fine) and select **Extract**.
5. After extraction, you should see a folder containing seven files (two `.xlsx` and five `.docx`).

### Step 2: Upload the files to OneDrive

1. Open **Microsoft Edge** and navigate to [https://www.office.com](https://www.office.com).
2. Sign in with your **Microsoft 365** credentials.
3. Select **Apps** in the left navigation pane, then select **OneDrive**.
4. In OneDrive, select **+ New** in the top-left corner, then select **Folder**. Name it **USLegal-ResourceFiles** and press **Enter**.
5. Open the **USLegal-ResourceFiles** folder you just created.
6. Select **+ New** > **Files upload** (or drag and drop the files directly into the browser window).
7. In the File Explorer window that appears, navigate to the folder where you extracted the zip file. Select **all seven files** and then select **Open**.
8. Wait for the upload to complete. You should see a confirmation message at the bottom of the screen: **Uploaded 7 items**.
9. Verify that all seven files appear in your **USLegal-ResourceFiles** folder:
    - USLegal_Account_List.xlsx
    - USLegal_Competitor_Pricing.xlsx
    - USLegal_Company_Overview.docx
    - USLegal_Service_Catalog.docx
    - USLegal_Client_Account_History.docx
    - USLegal_Prospect_Meeting_Transcript.docx
    - USLegal_Client_QBR_Meeting_Transcript.docx

### Step 3: Add files to your Most Recently Used list

Copilot in some apps can only reference files from your **Most Recently Used (MRU)** list. Open these three files so Copilot can find them during the exercises:

1. In OneDrive, select **USLegal_Account_List.xlsx** to open it in Excel for the web. Once it loads, close the tab.
2. Select **USLegal_Company_Overview.docx** to open it in Word for the web. Once it loads, close the tab.
3. Select **USLegal_Client_Account_History.docx** to open it in Word for the web. Once it loads, close the tab.

> **Important:** Microsoft 365 Copilot can only work with files saved to OneDrive. Files stored locally on your PC must be uploaded to OneDrive for Copilot to access them.

You're ready to begin the exercises.
